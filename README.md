# C Systems Programming – Stadium Management

This project implements a system for managing stadium seating using low-level programming in C.

## Features

* Custom data structures for managing stands and seating
* Bit-level operations for efficient seat representation
* Manual memory management using malloc/calloc/free
* File parsing and input validation
* Modular design across multiple components

## Technologies

* C
* Low-level memory management
* Bit manipulation
* File I/O

## Description

The system simulates seat allocation and management in a stadium environment, focusing on efficiency and correctness.
